Este PR añade las rutas iniciales y las rutas de la historia de usuario de corte de caja a nuestro servidor Express. Ahora tenemos dos rutas principales:
- La ruta raíz ('/') que devuelve un mensaje "Soy yo".
- La ruta '/api/cortecaja' que devuelve un mensaje "Ya responde".

He creado una clase `IndexRoutes` que configura la ruta raíz y devuelve el mensaje "Soy yo".
2) He creado una clase `cortecajaRoutes` que configura la ruta '/api/cortecaja' y devuelve el mensaje "Ya responde".
3) Se ha actualizado el archivo `index.ts` para utilizar estas rutas.

### Pruebas Realizadas
- Se ha probado accediendo a `http://localhost:3000` y se ha verificado que muestra "Soy yo".
- Se ha probado accediendo a `http://localhost:3000/api/cortecaja` y se ha verificado que muestra "Ya responde".

### Cómo Probarlo
1. Ejecuta el servidor con `npm run dev` y `npm run build`.
2. Abre tu navegador y accede a `http://localhost:3000` para ver la ruta inicial.
3. Abre tu navegador y accede a `http://localhost:3000/api/cortecaja` para ver la ruta.

Además, agregué la carpeta database con el archivo database.sql y agregué el script de la base de datos.
